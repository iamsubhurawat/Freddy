Freddy

Talk with your own pdf file

Freddy is a voice assistant that helps you to talk with any pdf file you upload. You can do question answering with freddy on the basis of the content of the uploaded pdf file.

Try Freddy

Play around with Freddy to know the information that your pdf file is containing without even reading the whole pdf file.

Technologies on which Freddy is built.

Groq

Groq is an LLM used for generating human-like responses for the user query.

Learn more about Groq
Deepgram API

Deepgram API is used for transcription and text-to-speech generation purposes.

Learn more about Deepgram
Faiss

Faiss is a vector database used to store the embeddings for the text from the documents.

Learn more about Faiss
Langchain

Langchain is a framework used to connect language models with external data sources.

Learn more about Langchain
Mongodb

Mongodb implemented with flask is used in backend to fetch & store data from the server.

Learn more about Mongodb
